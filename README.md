# OpenGL 3D game
## Requirements
* _glew_
* _glfw_
* _CMake_
## Installation
* Clone repository
```
git clone https://github.com/mikolajdlugolecki/game.git
```
* Create and go into _build_ directory
```
mkdir build && cd build
```
* Create CMake dependencies
```
cmake .. && cmake --buil .
```
* Build the final project
```
make
```
* Finally, run the game
```
./Game
```
